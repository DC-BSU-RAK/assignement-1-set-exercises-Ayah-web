A Quick Guide to HTML & CSS

HTML Tags

<!DOCTYPE html>

This defines the document's type and the version of HTML to be used.

<!DOCTYPE html>

<html>

This is the root element of an HTML document.

<html lang="en"></html>

<head>

This contains meta-information about the document, such as the title of the webpage and links to stylesheets.

<head>
    <title>Page Title</title>
</head>

<body>

This contains the visible content of the webpage.

<body>
    <h1>Hello, World!</h1>
</body>

<h1> to <h6>

This defines headings of various types, with <h1> being the biggest and most important.

<h1>Heading 1</h1>
<h2>Heading 2</h2>
<h3>Heading 3</h3>

<p>

This defines a paragraph of text.

<p>This is a paragraph.</p>

<a>

This defines a hyperlink.

<a href="https://example.com">Visit Example</a>

<img>

This embeds an image into the document.

<img src="image.jpg" alt="Description">

<ul> and <li>

This defines an unordered list and list items.

<ul>
    <li>Item 1</li>
    <li>Item 2</li>
</ul>

<div>

This is an empty container for grouping and styling content (does not alter text in html).

<div class="container">
    <p>Content goes here.</p>
</div>

CSS Properties

color

This sets the text color.

color: #00ff00;

background-color

This sets the background color.

background-color: #000;

font-family

This defines the font for a text.

font-family: "Courier New", monospace;

font-size

This sets the size of a text.

font-size: 16px;

margin

this sets the margin around an element.

margin: 10px;

padding

This sets the padding inside an element.

padding: 10px;

border

This defines a border around an element.

border: 1px solid #00ff00;

text-align

This aligns text horizontally.

text-align: center;
